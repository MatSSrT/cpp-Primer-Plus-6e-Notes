1.初始化将赋值和声明合并在一起。也可以使用字面值常量来初始化。也可以直接使用表达式来初始化变量。
```
int variable00 = value;
int variable01 = value1 + value2;
```

2.如果不对函数内部定义的变量进行初始化，该变量的值不确定的。

3.将大括号初始化用于单值变量。

4.在大括号内也不包含任何东西。此时变量初始化为零。

5.变量的初始化的问题，有助于防范类型转换错误。
