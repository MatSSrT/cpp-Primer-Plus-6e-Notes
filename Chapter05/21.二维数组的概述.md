## 什么是二维数组？
二维数组就是一个表格 - - - - - 有 `数据行` 和 `数据列` 。

C++不提供二维数组类型。但是可以创建每个元素本身都是数组的数组。

例如：
```
int maxtemps[4][5];
```
![]()

## 访问数组元素
![]()

第一个下标 - - - `表示行` ，第二个下标 - - - `表示列`

要打印数组的所有内容，使用for循环来改变行和列。
```
for(int row = 0; row < 4;row++)
{
    for(int col = 0; col < 5;++col)
        cout<<maxtemps[row][col]<<"\t";
    cout<<endl;
}
```
