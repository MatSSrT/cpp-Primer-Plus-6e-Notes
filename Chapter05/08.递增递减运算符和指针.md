## 递增/递减运算符和指针
递增运算符用于指针和基本变量。

将递增 / 递减运算符用于指针时，把指针增加其指向的数据类型占用的字节数。

```
 double arr[5] = {2.3,7.5,3.7,6.5,7.8};
 double *pt = arr;
 ++pt;
```

## 前缀/后缀递增 / 递减运算符的优先级
前缀递增、前缀递减和解除引用运算符的优先级相同，从左往右的方式进行结合。

后缀递增和后缀递减的优先级相同，但是比前缀运算符的优先级高，从左往右的方式进行结合。
